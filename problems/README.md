# DFS와 BFS 문제 모음

DFS와 BFS를 집중 훈련할 수 있도록 각 알고리즘 유형별로 10개씩, 총 20개의 문제를 정리했습니다. 모든 문제 파일에는 한글 설명과 함께 동작하는 파이썬 풀이가 포함되어 있습니다.

## DFS 문제 목록
1. `island_count.py`: 2차원 격자에서 섬의 개수 세기
2. `max_region.py`: 가장 큰 연결 영역 넓이 계산
3. `word_search.py`: 격자에서 단어 찾기
4. `permutations.py`: 숫자 목록의 모든 순열 생성
5. `combinations.py`: 조합 생성하기
6. `target_sum.py`: 부호 조합으로 목표 합 만들기
7. `maze_paths.py`: 미로 경로 수 세기
8. `all_paths.py`: DAG에서 모든 경로 나열하기
9. `tree_path_sums.py`: 이진트리 루트-리프 경로 합 구하기
10. `restore_ip.py`: 문자열에서 가능한 IP 주소 복원하기

## BFS 문제 목록
1. `maze_shortest_path.py`: 미로 최단 거리 구하기
2. `knight_moves.py`: 체스판 나이트 최소 이동
3. `tree_min_depth.py`: 이진트리 최소 깊이 찾기
4. `rotting_oranges.py`: 오렌지 썩는 시간 계산
5. `word_ladder.py`: 단어 사다리 최단 단계
6. `level_order.py`: 이진트리 레벨 순회 결과 구하기
7. `graph_shortest_steps.py`: 그래프 최단 이동 횟수
8. `matrix_distance.py`: 0에 대한 최소 거리 행렬
9. `spread_time.py`: 그래프 확산 완료 시간 측정
10. `open_lock.py`: 자물쇠 최소 회전 수 계산

각 문제는 `tests/test_problems.py`에서 제공하는 단위 테스트로 검증할 수 있습니다.
